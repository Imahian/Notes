# Certifications

Study notes for offensive security certifications — organized by vendor and difficulty level.

## eLearnSecurity / INE

| Cert | Full Name | Focus |
|------|-----------|-------|
| [eJPT](eJPT/) | Junior Penetration Tester | Network basics, enumeration, exploitation fundamentals |
| [eCPPT](eCPPT/) | Certified Professional Penetration Tester | Web, network, post-exploitation, pivoting |
| [eWPT](eWPT/) | Web Application Penetration Tester | OWASP Top 10, manual web exploitation |
| [eWPTX](eWPTX/) | Web Application Penetration Tester eXtreme | Advanced web: deserialization, XXE, JWT, race conditions |
| [eMAPT](eMAPT/) | Mobile Application Penetration Tester | Android/iOS static and dynamic analysis |
| [eCIR](eCIR/) | Certified Incident Responder | DFIR, log analysis, threat hunting |
| [eCTHP](eCTHP/) | Certified Threat Hunting Professional | Threat hunting, SIEM, IOC detection |
| [eCDFP](eCDFP/) | Certified Digital Forensics Professional | Disk, memory, network forensics |
| [eEDA](eEDA/) | Exploit Development Associate | Shellcode, stack overflows, SEH, DEP/ASLR bypass |
| [ICCA](ICCA/) | INE Certified Cloud Associate | Cloud security fundamentals — AWS, Azure, GCP |

## Difficulty Reference

```
eJPT → eWPT → eCPPT → eWPTX
           ↘ eMAPT
           ↘ eEDA → (OSCP level)
```

## Study Strategy

1. Read the syllabus and map to existing knowledge gaps
2. Labs before theory — hands-on first, read docs to fill gaps
3. Write notes as you go — teaching forces understanding
4. Attempt the exam only after passing all practice labs
