# HackTheBox

[hackthebox.com](https://hackthebox.com) — Competitive hacking platform with real machines, challenges, and structured learning paths.

## Sections

| Section | Description |
|---------|-------------|
| [Machines](MACHINES/) | Linux and Windows full compromise chains (foothold → root) |
| [Academy](ACADEMY/) | Structured modules from beginner to advanced |

## Connection

```bash
# Download your .ovpn from HTB → Connect → VPN
sudo openvpn <username>.ovpn
```

Machines are reachable at `10.10.x.x` after VPN connects. Your tun0 IP is your attacker address.

## Difficulty Rating

Machines are rated Easy / Medium / Hard / Insane by the community. Ratings reflect initial foothold difficulty, not necessarily the full chain.
