# Bug Bounty Hunter Path

HTB Academy structured path covering the full bug bounty workflow — from recon and web exploitation to writing professional reports.

| Module | Focus |
|--------|-------|

| [JavaScript Deobfuscation](JavaScript%20Deobfuscation/) | Decoding and reversing obfuscated JS |

| [SQLMap Essentials](SQLMap%20Essentials/) | Automated SQLi with sqlmap |

| [File Upload Attacks](File%20Upload%20Attacks/) | Extension bypass, webshells, polyglots |

| [Web Attacks](Web%20Attacks/) | IDOR, XXE, SSRF, Open Redirect |
