# HTB Academy

[academy.hackthebox.com](https://academy.hackthebox.com) — Structured learning platform by HackTheBox with guided modules, labs, and skill paths.

## Paths Covered

| Path | Level | Focus |
|------|-------|-------|
| [Bug Bounty Hunter](Bug%20Bounty%20Hunter/) | Intermediate | Web vulnerabilities, recon, reporting |

## Module Format

Each module follows: **theory → interactive labs → skills assessment**. Labs run in browser-based Docker environments or dedicated VMs — no VPN required.

| 3 | Advanced exploitation |
