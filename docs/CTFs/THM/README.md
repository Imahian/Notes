# TryHackMe

[tryhackme.com](https://tryhackme.com) — Guided learning platform with browser-accessible machines, CTF rooms, and structured paths for beginners through intermediate practitioners.

## Features

- **No VPN required** for most rooms — machines run in-browser via AttackBox
- **Learning paths** with curated room sequences
- **Hints and walkthroughs** available for all rooms
- **Streak and gamification** system for daily practice

## Connection (when using own Kali)

```bash
# Download OpenVPN config from TryHackMe → Access
sudo openvpn <username>.ovpn
# Target machines at 10.10.x.x
```

## Room Types

| Type | Description |
|------|-------------|
| Free | Open access, no subscription |
| Premium | Requires subscription |
| CTF | Competition-style, no hints |
| Learning | Guided with questions and answers |

## Key Paths

- **Pre-Security** — absolute beginner fundamentals
- **Complete Beginner** — Linux, web, networking, Metasploit
- **Jr Penetration Tester** — aligned with eJPT / PJPT level
- **SOC Level 1** — Blue team, log analysis, threat detection
