# CTFs

Writeups and notes for Capture The Flag competitions, wargames, and hacking labs — organized by platform.

| Platform | Type | Focus |
|----------|------|-------|
| [HackTheBox](HTB/) | Labs + Academy | Machines, challenges, structured paths |
| [OverTheWire](OTW/) | Wargames | Linux, web, crypto, binary exploitation |
| [TryHackMe](THM/) | Guided rooms | Beginner to intermediate |
| [VulnHub](VHB/) | Offline VMs | Full penetration test practice |

## Methodology

Every writeup follows a consistent structure:

1. **Reconnaissance** — port scan, service enumeration, OSINT
2. **Foothold** — vulnerability identification and initial shell
3. **Enumeration** — internal discovery, credential hunting
4. **Privilege escalation** — sudo, SUID, cron, kernel, token abuse
5. **Flags** — proof screenshots and hashes
